# 常用命令（Windows / PowerShell）

## 后端
- 进入后端目录：`cd backend`
- 启动后端：`mvn spring-boot:run`
- 运行测试：`mvn test`

## 前端
- 进入前端目录：`cd frontend`
- 安装依赖：`npm install`
- 启动开发：`npm run serve`
- 构建：`npm run build`
- Lint：`npm run lint`

## draw.io 本地静态服务
- `npx serve -l 8085`

## 常用工具
- 列目录：`Get-ChildItem`（或 `ls`）
- 搜索文本：`rg "pattern" path`
- 查找文件：`rg --files`
